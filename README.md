# OneLink Messenger

OneLink — это веб‑мессенджер для общения, как альтернатива Telegram.

## Что реализовано

- список чатов с поиском;
- создание нового чата;
- отправка сообщений;
- имитация автоответа от OneLink Bot;
- сохранение чатов и сообщений в `localStorage`;
- светлая и тёмная тема.

## Запуск

Откройте `index.html` в браузере.

Или запустите локальный сервер:

```bash
python3 -m http.server 8080
```

После этого откройте в браузере:

`http://localhost:8080`
