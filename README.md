# OneLink Messenger

OneLink — это веб‑мессенджер для общения, как альтернатива Telegram.

## Что реализовано

- регистрация/вход по электронной почте с кодом подтверждения;
- получение 6-значного кода (демо-режим для локального запуска);
- список чатов с поиском;
- создание нового чата;
- отправка сообщений;
- имитация автоответа от OneLink Bot;
- сохранение чатов и сессии в `localStorage`;
- светлая и тёмная тема;
- кнопка **«Скачать Android (.apk)»** в интерфейсе.

## Как работает подтверждение по email

1. Введите email в окне входа.
2. Нажмите «Получить код».
3. Введите 6-значный код и подтвердите вход.

> Сейчас это демо-режим: код отображается на экране (для локального теста без SMTP-сервера).

## Как работает скачивание .apk

Кнопка скачивает файл по пути:

`downloads/onelink.apk`

Чтобы это работало, поместите ваш собранный Android APK в папку `downloads/` с именем `onelink.apk`.

## Запуск

Откройте `index.html` в браузере.

Или запустите локальный сервер:

```bash
python3 -m http.server 8080
```

После этого откройте в браузере:

`http://localhost:8080`
